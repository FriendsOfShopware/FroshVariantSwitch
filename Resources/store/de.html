Dieses Plugin für Shopware erweitert die Funktionalität, um die zuvor ausgewählte Variante eines Produkts in den Ansichten Warenkorb und Kasse zu ändern.

Verfügbare Varianten können durch Anklicken von "Variante ändern" in der Cart/Checkout-Ansicht sowie im Off-Canvas Warenkorb ausgetauscht werden. Es wird ein Modal eröffnet, das alle relevanten Informationen enthält. Sowohl die Produktkonfiguration als auch die Menge können an dieser Stelle geändert werden.

Der Benutzer muss den Artikel nicht mehr aus dem Warenkorb entfernen, den gewünschten Artikel wiederfinden und stattdessen eine weitere Variante von der Artikelseite hinzufügen.

Das Plugin wird von der Github Organization FriendsOfShopware entwickelt.  Maintainer des Plugins ist:  David Neustadt
